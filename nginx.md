# nginx Note
#### 启动  
```
brew services start nginx
```
#### 关闭
```
brew services stop nginx
```
#### 查看是否安装成功
```
http://localhost:8080 
```
#### nginx安装文件目录
```
/usr/local/Cellar/nginx
```
#### nginx配置文件目录
```
/usr/local/etc/nginx
```
#### confing文件目录
```
/usr/local/etc/nginx/nginx.conf
```
#### 重新加载配置|重启|快速停止|安全关闭
```
nginx -s reload|reopen|stop|quit
```
